0. Переходимо в обрану папку в командній строці
    cd назва_папки (без пробілів)
1. створимо віртуальне оточення в обраній папці
    python3 -m venv venv (myenv, або venv або як завгодно назвати нове віртуальне оточення)
2. Активуємо віртуальне оточення
    source venv/bin/activate
3. Введемо наступну команду для встановлення стороннього пакета colorama
    pip3 install colorama
4. Перевіримо які пакети встановленні в віртуальному середовищі
    pip3 list
5. Створимо файл із залежностями 
    pip3 freeze > requirements.txt
6. Щоб встановити пакети, перелічені у requirements.txt, використовуйте команду:
    pip3 install -r requirements.txt
7. виконаємо деактивацію віртуального оточення
    deactivate
